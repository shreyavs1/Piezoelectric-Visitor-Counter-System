# Piezoelectric Visitor Counter System

## Overview

The **Piezoelectric Visitor Counter System** is a Linear Integrated Circuits project that utilizes the piezoelectric effect to convert mechanical energy generated from footsteps into electrical energy. The generated electrical signal is conditioned using analog circuits and used to power a visitor counting system.

The project demonstrates sustainable energy harvesting while providing an intelligent solution for real-time visitor monitoring in commercial environments such as shopping malls, airports, railway stations, and public buildings.

---

## Problem Statement

Develop a visitor counting system powered by **piezoelectric energy harvesting**, where electrical energy generated from human footsteps is utilized to detect and count visitors efficiently.

## Working Principle

1. A person steps on a piezoelectric sensor.
2. Mechanical pressure generates electrical voltage.
3. The voltage is buffered using an operational amplifier (Voltage Follower).
4. Noise is filtered and signal integrity is improved.
5. The processed signal is sent to the visitor counter.
6. The visitor count is displayed on a seven-segment display.
7. Data can be stored for monitoring and analytics.
